# COS30019-Intro-to-Artificial-Intelligence

## How to run search?

Format : `python search.py tests/<test_file> <method>`

```bash
# Change Directory
cd tree-based-search

# Search with BFS
python search.py tests/PathFinder-test.txt BFS

# Search with DFS
python search.py tests/PathFinder-test.txt DFS
```
